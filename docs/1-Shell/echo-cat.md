`echo` 仅仅是把其参数按照字符串的形式原样输出

`cat` 参数是文件，会将文件内容进行输出，当然其名源于 con`cat`enate，见名知意，其可接受多个参数，会将所有参数文件的内容按顺序进行输出

从上图可以看出，`cat` 可以以 stream 的形式接收输入，进而输出
